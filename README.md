# AetherPy
# 🐍 Python Learning Projects

A collection of Python projects and learning exercises — from beginner scripts to practical applications like face recognition and user authentication.

---

## 📁 Repository Structure

```
├── learning-lessons/       # Python basics and practice scripts
├── calculator/             # Simple calculator app
├── face-recognition/       # Face detection & recognition
└── login-system/           # User login & authentication
```

---

## 📂 Projects Overview

### 📘 Learning Lessons
Beginner-friendly Python scripts covering core concepts such as:
- Variables, data types, and operators
- Conditionals and loops
- Functions and modules
- File handling and more

### 🧮 Calculator
A simple command-line calculator that performs basic arithmetic operations — addition, subtraction, multiplication, and division.


### 👤 Face Recognition
A Python project using computer vision to detect and recognize faces from images or a webcam feed.

**Libraries used:** `opencv-python`, `face_recognition` / `deepface`

### 🔐 Login System
A basic authentication system with:
- User registration
- Password validation
- Login/logout functionality

---

## 🛠️ Requirements

```bash
pip install opencv-python face_recognition numpy
```

> Some projects may have their own dependencies. Check individual folders for any `requirements.txt`.

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/AA-TECH-PROJECTS/AetherPy.git
   cd AetherPy
   ```

2. **Navigate to any project folder**
   ```bash
   cd face-recognition
   ```

3. **Run the script**
   ```bash
   python main.py
   ```
